OpenClaw Browser Relay

Local relay server port (default: 18792)
Set this to match your Gateway auth.token
Automatically re-attach tabs after page refresh, extension restart, or connection issues

How to use

  1. Make sure OpenClaw Gateway is running with browser relay enabled
  2. Set the Relay Port to match your Gateway configuration
  3. Enter your Gateway Token (from Gateway auth settings)
  4. Click the extension icon on any tab to attach/detach
  5. When attached, OpenClaw can automate that tab
